# WiGuard AI

WiGuard AI is a lightweight intrusion detection system that analyzes Wi-Fi metadata to detect suspicious activity in real-time.

## 🚀 Features
- Real-time monitoring
- Metadata-based anomaly detection
- Lightweight and efficient
- Privacy-preserving (no payload inspection)

## 🧠 Technologies Used
- React + TypeScript
- Vite
- Tailwind CSS

## ▶️ How to Run
```bash
npm install
npm run dev
